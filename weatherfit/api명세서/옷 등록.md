# 옷 등록

완료: Yes
카테고리: 옷
Method: POST
URL: /clothes
설명: 옷 사진과 카테고리들을 저장
설계 여부: Yes

### Request

**Header** 

| **key** | **type** | **required** | **description** |
| --- | --- | --- | --- |
| X-DEVICE-ID | String | Y | 디바이스 UUID
 |

```json
X-DEVICE-ID : a-2356
```

**Body**

| key | type | required | description |
| --- | --- | --- | --- |
| `image` | `multipartFile` | Y | 옷 사진 |
| `data` | `ClothesRegisterRequest` | Y | 사진 외의 request |

**ClothesRegisterRequest**

| key | type | required | description |
| --- | --- | --- | --- |
| `category` | `enum` | Y | 대분류 카테고리 |
| `subCategory` | `enum` | Y | 소분류 카테고리 |

**Request Example**

```java
POST /clothes
```

### Response

| key | type | 설명 |
| --- | --- | --- |
| `id` | `Long` | 옷 번호 |
| `image` | `string` | 옷 사진 링크 |
| `category` | `enum` | 대분류 카테고리 |
| `category_sub` | `enum` | 소분류 카테고리 |

**Example**

```jsx
{
  "id": 1,
  "image": "http://slslsl",
  "category": "top",
  "category_sub": "shirt"
}
```
