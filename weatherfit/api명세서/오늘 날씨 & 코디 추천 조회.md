# 오늘 날씨 & 코디 추천 조회

완료: Yes
카테고리: 코디
Method: GET
URL: /today?latitude=37.5&longitude=126.9
설명: 오늘 날씨 조회 및 현재 날씨에 해당하는 코디들 조회 (코디 등록 최신순)
설계 여부: Yes

### Request

**Header** 

| **key** | **type** | **required** | **description** |
| --- | --- | --- | --- |
| X-DEVICE-ID | String | Y | 디바이스 UUID
 |

**PathParameter**

| key | type | required | description |
| --- | --- | --- | --- |
| `latitude` | `Double` | N | 위도 |
| `longtitude` | `Double` | N | 경도 |
| `loc` | `String` | N | 위치명(서울) |

**Request Example**

```java
GET /today?latitude=37.5&longitude=126.9?loc=Seoul
```

### Response

| key | type | 설명 |
| --- | --- | --- |
| region | `String` | 지역 |
| temperature | `Double` | 온도 |
| weather | `String` | 날씨 |
| outfits | `List<OutfitsResponse>` | 추천 코디들 |

**OutfitsResponse**

| key | type | 설명 |
| --- | --- | --- |
| id | `Long`  | 코디 번호 |
| thumbnail | `String`  | 코디 이미지 |

**Example**

```jsx
{
  "region": "Seoul",
  "temperature": 20.5,
  "weather": "Cloud",
  "outfits": [
	  {
		  "id": 2,
		  "thumbnail": "http://lodl",
	  },
	  {
		  "id": 1,
		  "thumbnail": "http://lodl",
	  },
  ]
}
```
