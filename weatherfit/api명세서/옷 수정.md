# 옷 수정

완료: Yes
카테고리: 옷
Method: PUT
URL: /clothes/{clothesId}
설명: 등록한 옷의 카테고리 수정 (변경된 필드만 담아 보내주세요)
설계 여부: Yes

### Request

**Header** 

| **key** | **type** | **required** | **description** |
| --- | --- | --- | --- |
| X-DEVICE-ID | String | Y | 디바이스 UUID
 |

```json
X-DEVICE-ID : a-2356
```

**PathVariable**

| key | type | required | description |
| --- | --- | --- | --- |
| `clothesId` | `Long` | Y | 옷 번호 (PK) |

**Body**

| key | type | required | description |
| --- | --- | --- | --- |
| `data` | `ClothesRegisterRequest` | Y | 옷 정보  |

**ClothesRegisterRequest**

| key | type | required | description |
| --- | --- | --- | --- |
| `category` | `enum` | N | 대분류 카테고리 |
| `category_sub` | `enum` | N | 소분류 카테고리 |

**Request Example**

```java
PUT /clothes/{clothesId}
```

### Response

| key | type | 설명 |
| --- | --- | --- |
| `id` | `Long` | 옷 번호 |
| `image` | `string` | 옷 사진 링크 |
| `category` | `enum` | 대분류 카테고리 |
| `category_sub` | `enum` | 소분류 카테고리 |

**Example**

```jsx
{
  "id": 1,
  "image": "http://slslsl",
  "category": "top",
  "category_sub": "shirt"
}
```
