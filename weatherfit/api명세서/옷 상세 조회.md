# 옷 상세 조회

완료: Yes
카테고리: 옷
Method: GET
URL: /clothes/{clothesId}
설명: 등록한 옷 상세 조회
설계 여부: Yes

### Request

**Header** 

| **key** | **type** | **required** | **description** |
| --- | --- | --- | --- |
| X-DEVICE-ID | String | Y | 디바이스 UUID
 |

```json
X-DEVICE-ID : a-2356
```

**PathVariable**

| key | type | required | description |
| --- | --- | --- | --- |
| `id` | `Long` | Y | 옷 번호 (PK) |

**Request Example**

```java
GET /clothes/1
```

### Response

| key | type | 설명 |
| --- | --- | --- |
| `id` | `Long` | 옷 번호 |
| `image` | `string` | 옷 사진 링크 |
| `category` | `enum` | 대분류 카테고리 |
| `category_sub` | `enum` | 소분류 카테고리 |

**Example**

```jsx
{
  "id": 1,
  "image": "http://slslsl",
  "category": "top",
  "category_sub": "shirt"
}
```
