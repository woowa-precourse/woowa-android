# 분류별 사용자 소유 옷 조회

완료: Yes
카테고리: 옷
Method: GET
URL: /clothes?category=top&sub=shirt&cursor=10&size=10
설명: 사용자가 등록한 옷들 카테고리 별 조회 (최신순)
설계 여부: Yes

### Request

**Header** 

| **key** | **type** | **required** | **description** |
| --- | --- | --- | --- |
| X-DEVICE-ID | String | Y | 디바이스 UUID
 |

```json
X-DEVICE-ID : a-2356
```

**PathParameter**

| key | type | required | description | default |
| --- | --- | --- | --- | --- |
| `category` | `string` | N | 대분류 카테고리 | top |
| `sub` | `string` | N | 소분류 카테고리 | - |
| `cursor` | `Long` | N | 마지막 옷 id | - |
| `size` | `Int` | N | 조회 개수 | 20 |

**Request Example**

```java
GET /clothes?category=top&sub=shirt&cursor=10&size=10
```

### Response

| key | type | 설명 |
| --- | --- | --- |
| `clothes` | `List<ClothesResponse>` | 옷 리스트 |
| `category` | `enum` | 대분류 카테고리 |
| `category_sub` | `enum` | 소분류 카테고리 (빈 값일 때는 전체) |

**ClothesResponse**

| key | type | 설명 |
| --- | --- | --- |
| `id` | `Long` | 옷 번호 |
| `image` | `string` | 옷 사진 링크 |

**Example**

```jsx
{
  "clothes": [
	  {
		  "id": 1,
		  "image": "http://lodl",
	  },
	  {
		  "id": 1,
		  "image": "http://lodl",
	  }
  ],
  "category": "top",
  "category_sub": "", // 전체
}
```
